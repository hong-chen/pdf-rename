# pdf-rename
--------
__About__

Auto rename pdf files (books, papers) using the format "author-year-title".

--------

__Requirements__

- Python environment: Python 3.6.0 :: Anaconda 4.3.1 (x86_64)

- [pdfminer.six](https://github.com/pdfminer/pdfminer.six)
  `pip install pdfminer.six`
